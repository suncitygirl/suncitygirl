Examples

  • Mishka — адаптивный резиновый сайт. Главная страница, страницы с каталогом товаров и формой заказа
  • Pink — адаптивный резиновый, одностраничный. На страничке реализована карусель, полностью на HTML/CSS
  • Simon Game — простенькая игра (wiki) из двухсот строк на JS.
  • Dran'n'drop TODO demo - реализация Drag'n'Drop. Только JS, только события мыши. Даты сортируются по порядку, здачи можно вкладывать друг в друга. Новая задача по умолчанию добавляется в последний день, можно ее перенести.
